Appendix A

Software Product

Product Name:Modgen

Product Description:

Modgen is a microsimulation programming language product supporting the creation, maintenance and documentation of dynamic microsimulation models. The product has two main components, each with its own associated installation program:

  • Modgen Prerequisites, which contains the software and documentation necessary to run a Modgen-created model on a given machine
  • Modgen itself, which contains the software and documentation necessary to modify existing models or develop new models

Documentation:

  • Modgen Developer's Guide
  • Modgen Guide du concepteur
  • Guide to the Modgen Visual Interface
  • Guide de l'interface visuelle de Modgen
  • Material from various technical workshops
  • Sample models obtained from Statistics Canada
