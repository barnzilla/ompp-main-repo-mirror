Annexe A

Produit logiciel

Nom du produit :Modgen

Description du produit :

Modgen est un produit de langage de programmation de microsimulation qui permet de créer, de maintenir et de documenter des modèles de microsimulation dynamique. Il compte deux composants principaux possédant chacun son propre programme d'installation :

  • Composants Préalables à Modgen, qui comprend le logiciel et la documentation connexe nécessaires à l'exécution d'un modèle créé à l'aide de Modgen sur un appareil donné;
  • Modgen, qui contient le logiciel et la documentation connexe requis pour modifier des modèles existants ou créer de nouveaux modèles.
