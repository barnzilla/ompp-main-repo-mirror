To build openM++ libraries and omc compiler use:

  docker run .... <image-name> do-build cpp

Environment variables:
  set OM_SKIP_OPENM=yes              (default: no)
  set OM_SKIP_MODELS=yes             (default: no)
  set OM_BUILD_CONFIGS=Release Debug (default: Release)
  set OM_BUILD_PLATFORMS=Win32 x64   (default: Win32)
  set OM_MSG_USE=MPI                 (default: EMPTY)
  set MODEL_DIRS=modelOne NewCaseBased NewTimeBased NewCaseBased_bilingual NewTimeBased_bilingual IDMM OzProj OzProjGen RiskPaths

To build openM++ tools:

  docker run .... <image-name> do-build tools

Environment variables:
  set OM_SKIP_GO=yes   (default: no)
  set OM_SKIP_UI=yes   (default: no)
  set OM_SKIP_R=yes    (default: no)
  set OM_SKIP_PERL=yes (default: no)

To open cmd command prompt or Perl command prompt:

  docker run .... -it <image-name> cmd
  docker run .... -it <image-name> do-build perl
